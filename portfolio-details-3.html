Sudoku Puzzle Solver - Utilizing Depth First Search

Project information

  • Category: Algorithms
  • Associated With: Purdue University Global
  • Status: Complete
  • Repository: GitHub

This project features the creation of an automated tool designed to solve Sudoku puzzles on the website sudoku9x9.com. Utilizing Depth-First Search (DFS), a well-known algorithmic technique for exploring all possible configurations, the tool efficiently finds solutions to even the most challenging puzzles. The script is written in JavaScript and is executed within the Tampermonkey userscript environment, allowing it to interact directly with the webpage elements. It initializes the Sudoku grid from the website, applies the DFS algorithm to explore possible number placements systematically, and fills in the puzzle with the correct answers.
Skills: Algorithm Implementation, JavaScript Programming, DOM Manipulation, Debugging, Web Automation, Logical Reasoning, Problem Solving
